下载 APK

点击上方按钮下载 APK 文件到你的 Android 手机。

2

允许安装

在手机「设置」→「安全」中开启「允许安装未知来源应用」。

3

安装并打开

打开下载的 APK 文件，点击安装。完成后打开 Solace 即可使用。

4

配置 API Key

首次使用需要配置 API 密钥。推荐 SiliconFlow，国内可用且新用户有免费额度。
