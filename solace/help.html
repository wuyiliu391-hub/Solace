入门指南

如何安装？

  1. 下载 APK 文件到你的 Android 手机
  2. 在手机「设置」→「安全」中开启「允许安装未知来源应用」
  3. 打开下载的 APK 文件，点击安装

如何配置 API？

首次使用需要配置 API 密钥：

  1. 前往 SiliconFlow 注册账号并获取 API Key
  2. 在 Solace 的「设置」→「API 配置」中填入 API Key
  3. 选择模型（推荐 minimax-m2.7 或 gpt-4o-mini）

如何创建角色？

  1. 点击底部「联系人」→「创建角色」
  2. 填写角色名称、性格、心愿等信息
  3. 保存后即可开始对话

常见问题

数据存储在哪里？

所有数据仅存储在你的设备本地 SQLite 数据库中，不上传至任何服务器。API Key 也仅保存在本地。

为什么需要 API Key？

AI 对话需要调用大语言模型的接口。我们不提供内置模型（不像 Lovemo 那样），这样你可以自由选择 API 服务商和模型，费用更透明。

应用闪退怎么办？

尝试以下步骤：

  1. 清除应用缓存（不删除数据）
  2. 检查是否有足够的存储空间
  3. 确保 Android 版本 >= 7.0
  4. 重新安装应用
