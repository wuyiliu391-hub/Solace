隐私政策

最后更新：2026 年 1 月 1 日 · 版本 1.0

Solace 是一款零后端、纯本地的 AI 陪伴应用。本隐私政策旨在说明我们如何保护你的隐私。

我们不收集的数据

  • 聊天内容和消息记录
  • 角色配置和个性化设定
  • API 密钥和认证信息
  • 任何形式的设备标识符（IDFA、GAID 等）
  • 崩溃诊断数据

本地存储

所有数据（聊天记录、角色信息、设置等）均存储在你的设备本地 SQLite 数据库中。我们不会将这些数据上传至任何服务器或第三方服务。

API 使用

AI 对话功能需要使用你自行配置的 API 密钥调用第三方 AI 服务。API 密钥仅保存在本地，不会上传到我们的服务器。

第三方服务

我们不使用任何第三方分析、统计或广告 SDK。应用内不包含 Facebook SDK、Firebase Analytics、Google Analytics 等追踪工具。

数据安全

应用支持数据加密存储。建议用户妥善保管设备，防止未经授权的访问。

未成年用户

Solace 适合 16 岁及以上用户使用。

变更通知

如本政策发生变更，我们将在应用内和网站上发布更新。

联系我们

如有任何隐私相关的问题，请通过应用内反馈渠道或邮件联系我们。
