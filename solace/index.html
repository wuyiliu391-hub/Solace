常见问题

你可能想知道的

Solace 需要联网吗？

需要联网调用 API，但所有聊天记录和个人数据都保存在本地设备上，不会上传到任何服务器。

我的数据安全吗？

非常安全。所有数据存储在本地，不会上传到任何服务器。API 密钥也仅保存在本地设备上。

需要什么 API 密钥？

需要兼容 OpenAI API 格式的接口地址和密钥，推荐使用 SiliconFlow（国内可用，新用户有免费额度）。

收费吗？

应用本身完全免费。你只需自行承担 AI API 调用的费用。

未成年人可以使用吗？

仅限 18 周岁及以上用户使用。根据《人工智能拟人化互动服务管理暂行办法》，本应用已实施身份证年龄验证。身份证仅在前端本地运算验证，不上传服务器，验证完成后立即清空，仅保留周岁数字用于功能限制。
