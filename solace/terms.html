用户协议

最后更新：2026 年 1 月 1 日 · 版本 1.0

欢迎使用 Solace！在使用本应用之前，请仔细阅读本用户协议。

1. 服务说明

Solace 是一款 AI 陪伴应用，提供基于大语言模型的对话服务。用户需自行配置 API 密钥，所有数据存储在本地设备上。

2. 用户资格

你必须年满 16 周岁才能使用本应用。使用应用即表示你确认自己符合上述年龄要求。

3. 用户责任

你同意：

  • 自行承担 API 调用费用
  • 妥善保管 API 密钥，不泄露给第三方
  • 不利用本应用从事违法活动
  • 不传播违法违规内容

4. 免责声明

AI 生成的内容可能存在不准确之处，仅供娱乐和参考，不构成专业建议。开发者不对 AI 生成的内容负责。

5. 知识产权

Solace 应用的代码、UI 设计等知识产权归开发者所有。用户创建的角色和对话内容归用户所有。

6. 协议变更

我们可能随时修改本协议。修改后的协议将在应用内和网站上发布。

7. 终止

你可以随时停止使用本应用。我们也可能在任何时候终止提供服务。

8. 法律适用

本协议适用中华人民共和国法律。
