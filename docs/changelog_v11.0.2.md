# v11.0.2 更新日志

## 🐛 Bug 修复

### 消息系统修复（重要）
- **修复用户发送消息不显示的问题**：正则表达式崩溃导致用户消息无法保存
- **修复消息归属混乱**：数据库迁移时 `isUser` 字段默认值导致所有消息被标记为 AI 消息
- **修复旧版本用户升级后消息显示异常**：添加自动修复逻辑，升级后自动修正消息归属

### 数据兼容性
- 旧版本用户升级不需要清除数据，修复自动执行
- 旧版本导出的数据导入后会自动修复

## 📝 技术改进

- 优化数据库迁移逻辑，确保新列添加时同步更新已有数据
- 使用 SharedPreferences 标记确保一次性修复逻辑不会重复执行
- 改进消息保存流程，添加内存保护机制
