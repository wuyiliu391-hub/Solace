# 📢 Solace v11.0.2 更新公告

## ⚠️ 重要提醒：请先备份再更新！

本次更新为**深度重构**，涉及消息系统核心逻辑。为确保您的数据安全，请务必在更新前完成以下步骤：

### 📦 备份方法
1. 打开 Solace → 设置 → 数据管理 → 导出数据
2. 将导出文件保存到安全位置（建议存到云盘或电脑）
3. 确认备份文件完整后，再进行更新

### 🔄 更新内容

#### 🐛 关键修复
- **修复用户消息不显示的问题**：部分用户反馈发送消息后聊天界面无显示，现已修复
- **修复消息归属混乱**：修复消息气泡方向错误（用户消息显示在左侧/AI侧）
- **修复旧版本数据兼容性**：升级后自动修复历史消息数据，无需手动操作

#### 🛠️ 技术改进
- 重构消息存储系统，提升稳定性
- 优化数据库迁移逻辑，确保数据完整性
- 改进消息保存流程，添加多重保护机制

### 📋 升级说明

| 场景 | 操作 |
|------|------|
| 新用户 | 直接安装使用，无需任何操作 |
| 旧版本用户（未导入数据） | 直接更新，自动修复 |
| 旧版本用户（已导入数据） | **强烈建议先备份**，然后更新，数据自动修复 |
| 已清除数据的用户 | 直接更新，无需操作 |

### ⚠️ 注意事项

1. **请勿在更新过程中关闭 app**
2. **更新后首次打开可能需要几秒钟完成数据修复**
3. **如遇问题，请使用备份文件恢复数据**
4. **更新后如消息显示异常，请尝试清除缓存后重启 app**
