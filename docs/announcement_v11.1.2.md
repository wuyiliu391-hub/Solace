# 📢 Solace v11.1.2 更新公告

## 🐛 修复中文乱码

本次更新修复了部分安卓设备上出现的中文乱码问题。优化了 HTTP 响应解码机制，确保中文内容正确显示。

### 修复内容
- **中文乱码**：优化 HTTP 客户端，支持 gzip 自动解压，解决中文显示为乱码的问题
- **繁简转换**：修复繁体到简体转换时的误转换问题
- **小模型回复**：减少小模型重复输出相同内容的情况

## ✨ 联网搜索基础

新增联网搜索基础设施，为后续开启供应商原生联网搜索做好准备。

### 当前状态
- 联网搜索功能已开发完成，暂未启用
- 后续将支持供应商原生联网搜索，无需额外配置即可使用
