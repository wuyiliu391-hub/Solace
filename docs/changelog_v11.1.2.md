# v11.1.2 更新日志

## 🐛 Bug 修复

### 中文乱码修复
- **修复安卓端中文乱码问题**：HTTP 响应解码优化，支持 gzip 自动解压
- **修复繁简转换异常**：优化繁体到简体的转换逻辑，避免误转换

### 消息系统修复
- **修复小模型重复回复**：优化消息去重逻辑，减少重复内容输出
- **修复搜索结果注入异常**：搜索结果为空时正确提示用户，不再继续角色扮演

## ✨ 新功能

### 联网搜索（已禁用）
- 新增联网搜索基础设施，支持 Bing 搜索和 Jina Search 兜底
- 为供应商原生联网搜索预留接口，后续可通过配置开启

## 📝 技术改进

- HTTP 客户端从 `http` 包迁移到 `dart:io` HttpClient，提升 gzip 兼容性
- 添加搜索调试日志，便于排查搜索失败问题
- 优化小模型联网搜索时的提示词，避免被角色人设淹没
