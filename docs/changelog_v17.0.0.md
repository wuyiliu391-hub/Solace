# Solace v17.0.0 Changelog

版本：17.0.0 (Build 277)
日期：2026-07-08

## 新功能

- 发现页面全新改版：支持左滑切换第二分页
- 新增娱乐专区：真心话大冒险、默契测试、心有灵犀、角色印象
- 旧版全部历史功能统一收纳至「更多娱乐」入口
- 设备操控测试版上线，新增锁屏专属交互模式
- 内置官方角色「老大」（项目作者），内置强制规范：严禁色情、违法、调教、低俗类相关内容
- 故事书入口迁移至发现页面，访问更便捷

## 仿生沉浸体验深度升级

- 消息持久化存储
- 角色情绪状态持久化
- 动态打字延迟效果
- 角色结构化特征存档
- 用户每日行为轨迹持久保存
- 增量记忆库二次迭代更新
- 情绪感知敏感度全面加深

## 优化

- 单聊界面优化：对话文本与旁白内容分隔展示，AI 输出文字统一采用蓝色字体

## Bug 修复

- 修复多款模式下内容分段异常、一次性完整刷屏输出的故障

## 已知问题

- 单聊界面对话文本与旁白分隔展示在部分场景下仍存在未知 BUG，后续版本持续修复优化
